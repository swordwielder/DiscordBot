How to install StockBot (with BetterDiscord)
How to install StockBot on
Step 1
Install StockBot
DOWNLOAD
Step 2
Join the StockBot Community server
JOIN
Step 4
Open discord then go in settings -> BetterDiscord -> Themes -> Open theme folder.
Step 5
Slide the
.theme.css
file you downloaded into the folder that just opened.
Step 6
Go back to Discord and do
ctrl+r
to reload it.
Step 6
Step 6: Go back in settings -> BetterDiscord -> Themes and enable the theme.
Step 7
You are done!
